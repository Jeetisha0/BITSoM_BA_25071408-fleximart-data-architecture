# Part 2: NoSQL Database Analysis (MongoDB)

- Evaluates limitations of relational databases for flexible product data
- Justifies the use of MongoDB for managing a diverse product catalog
- Explains benefits such as:
  - Flexible schema
  - Embedded documents
  - Scalability
- Discusses trade-offs of using MongoDB
- Uses a JSON-based product catalog dataset
- Demonstrates MongoDB operations including:
  - Loading data
  - Filtering queries
  - Aggregation for review analysis
  - Update operations
