# Part 3: Data Warehouse and Analytics

- Designs a data warehouse for historical sales analysis
- Implements a star schema with:
  - One fact table for sales
  - Dimension tables for date, product, and customer
- Explains design decisions and data flow in the warehouse
- Populates the warehouse with realistic sample data
- Writes OLAP queries to support:
  - Time-based drill-down analysis
  - Product performance evaluation
  - Customer segmentation
- Focuses on analytical reporting and business intelligence use cases
