# Part 1: Database Design and ETL Pipeline

- Implements a complete ETL pipeline for FlexiMart using raw CSV files
- Handles customer, product, and sales datasets with data quality issues
- Performs data cleaning such as:
  - Removing duplicate records
  - Handling missing values
  - Standardizing phone numbers, categories, and dates
- Loads cleaned data into a MySQL relational database
- Generates a data quality report summarizing ETL actions
- Documents the database schema and relationships
- Includes SQL queries to answer key business questions on:
  - Customer purchase history
  - Product sales analysis
  - Monthly sales trends
